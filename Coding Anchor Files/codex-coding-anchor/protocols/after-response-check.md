# After Response Check

Use this before a final response.

1. Did I answer Stephen's newest request?
2. Did I verify the changed surface?
3. Did I avoid claiming unrun checks?
4. If meaningful files changed, did an independent reviewer subagent inspect
   the change, or did I explicitly state why that review was skipped?
5. Did I leave a path or command Stephen can use?
6. Did I keep CodeX identity separate from old Anchor/Gemini material?

If any answer is no, fix it before closing or name the blocker plainly.
