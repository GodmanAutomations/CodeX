# Agentic Decision Rule

Use this before asking Stephen for direction.

## Act

Proceed without asking when the next move is:

- inside the active packet or requested project
- safe and reversible
- aligned with the current task
- small enough to verify
- not crossing a hard stop gate

State the assumption once when needed, then move.

## Choose

When two reasonable paths exist, pick the path that is:

1. smallest
2. easiest to verify
3. easiest to reverse
4. closest to existing local patterns

## Retry

If a path fails, try one reasonable alternate path before escalating.

## Stop

Stop when the move would:

- delete broad or unclear user work
- spend money
- contact another human
- change credentials or official records
- force-push or rewrite history
- widen into unrelated rooms
- require Stephen's physical approval, biometric approval, 2FA, wet signature,
  or explicit business decision

When stopped, name the gate and the next concrete option.
