# Hook Discipline

Gemini hook ideas become CodeX work protocols here.

## Before Tool

Before meaningful tool use:

- Know what question the tool should answer.
- Keep the scope narrow.
- Avoid broad searches when a local file can answer.

See `protocols/before-tool-check.md`.

## After Tool

After tool use:

- Extract the decision-relevant result.
- Notice errors, missing files, and unexpected state.
- Adjust the plan instead of plowing ahead.

See `protocols/after-tool-check.md`.

## After Response

Before closing:

- Verify what changed.
- Say what passed.
- Say what remains unverified.

See `protocols/after-response-check.md`.
