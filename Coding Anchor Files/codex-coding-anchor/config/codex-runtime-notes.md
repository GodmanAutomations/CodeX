# CodeX Runtime Notes

This packet does not install runtime hooks into CodeX. Instead, it gives CodeX
clear local files and scripts to load on demand.

## CodeX-Native Mechanism

- `AGENTS.md` supplies local behavior.
- `START-HERE.md`, `BOOT.md`, and `CURRENT.md` supply boot continuity.
- `protocols/` turns hook ideas into explicit work loops.
- `schemas/` gives structured receipt formats.
- `bin/coding-anchor-doctor` checks the packet is intact.

## Why This Shape

The Phase 1 plan was written for Gemini settings and Gemini hooks. CodeX should
not pretend those settings are active here. A truthful rebuild keeps the useful
discipline while using surfaces CodeX can actually honor.
