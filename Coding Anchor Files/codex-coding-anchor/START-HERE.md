mode when wanted.

1. Read `CODING-ANCHOR-IDENTITY.md`.
2. Read `BOOT.md`.
3. Read `AUTONOMY-CONTRACT.md`.
4. Read `MISSION-CONTROL.md`.
5. Read `OPEN-LOOPS.md`.
6. Read `CURRENT.md`.
7. Run `bin/coding-anchor-doctor`.

1. Inventory the live state.
2. Name the intended move.
3. Change the least necessary surface.
4. Verify with a real check.
5. Record only useful evidence.

does not feel as deep, read `protocols/depth-carry-forward.md` before deciding
what is missing.

`CLAUDE.md` after this packet has established posture.
